输出语句

echo

​ echo可以将紧跟其后一个或多个字符串、表达式、变量和常量的值输出到页面中，多个数据之间用”,”分割

print

print与echo用法相同，唯一区别是print一次只能输出一个值

print_r()

print_r()是PHP的内置函数，它可以输出任意类型的数据，如字符串、数组等。

var_dump()

var_dump()不仅可以打印一个或多个任意类型的数据，还可以获取数据的类型和元素个数

PHP标识符

在PHP程序开发中，经常需要自定义一些符号来标记一些名称，如变量名、函数名等，需要遵循一下规则：

（1）只能由字母、数字、下划线组成，不包括空格

（2）只能以字母或下划线开头的任意长度的字符组成

（3）用作变量名时，区分大小写

（4）由多个单词组成时用下划线分割

变量

变量概述

变量是用于临时存储值(数据)的容器，在程序运行中才存在

用于存储数字、文本、文本字符串、数组等

变量声明与赋值

php变量使用前不需要声明，直接$变量名使用即可，用=可对变量进行赋值

（2）传统赋值

（2）引用赋值

当$number改为100时，$result也会改为100，这是由于引用赋值的方式相当于给变量取一个别名，当一个变量的值发生变化时，另一个也随之变化

变量销毁
可变变量

可变变量的含义是指变量的名字有可以用另外一个变量来代替

实现在一个变量前多加一个美元符号

常量

常量定义

​ 常量是在脚本运行过程中值始终不变的量。

​ 常量一旦定义就不能被修改或重新定义

​ PHP中常量习惯用大写字母表示

​ 定义方法：

(1)define()函数

（2）const关键字

预定义常量

PHP提供了很多预定义常量，这些常量专门用于获取php中的信息，并且不允许开发人员修改。

数据类型

标准类型

​ a.bool型：只有true和false两个值

​ b.整型：可表示十进制、八进制、十六进制，用来表示整数。在前面 加上负号可以表示负数。

​ c.浮点型：表示小数的一种方法，有效位数为14位

​ d.字符串型
